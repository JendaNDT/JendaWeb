MeditApp

Offline-first PWA pro meditaci s procedurálně generovanými ambientními zvuky. Žádné předplatné, žádný cloud, žádné cookies.

Problém

Existující meditační aplikace jsou předplaceným softwarem napěchovaným analytikou. Většina obsahu je za paywallem, dlouhé sezení vyžadují účet, a všechny chtějí mít na pozadí běžící cloud spojení. Pro někoho, kdo si jen chce na pět minut sednout, je to overkill.

Řešení

  • PWA s instalací jedním klikem — žádný app store, žádné verzování
  • Ambientní zvuky generované procedurálně přes Web Audio API (déšť, oceán, šum lesa, čisté frekvence) — zero downloadu zvukových souborů
  • Service Worker pro plný offline režim; sezení se ukládají do IndexedDB
  • Řízené dýchání (4-7-8, box, koherentní) s vizuální oporou
  • Bez registrace, bez sledování, bez síťových requestů kromě úvodního loadu
